# Dice-simulator

Simulador de dados con estadísticas en tiempo real. Proyecto del Grupo 20 de Programación III — UTN FRBB.
 
## Qué hace?
 
Tirás dos dados y te muestra la suma obtenida. Durante la sesión lleva un registro de cuántas veces salió cada resultado posible (del 2 al 12), mostrando el conteo, el porcentaje y una barra de progreso por cada uno. También podés reiniciar todo con el botón de reset.
 
## Tecnologías usadas
 
- HTML5 para la estructura
- CSS3 con Flexbox para el layout y las animaciones
- JavaScript vanilla para la lógica y manipulación del DOM
- Git y GitHub para el trabajo colaborativo
